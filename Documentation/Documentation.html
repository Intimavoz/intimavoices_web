Installation

Installing Template

FTP Upload:

  • Open up your FTP manager and connect to your hosting
  • Browse to required directory (Normally public_html)
  • Upload the files inside HTML folder

Pages

Page details for HTML template

Pages are as follows:

  • index.html is Version 1 Home page
  • 404.html is 404 page
  • about.html is About Us Page
  • blog-grid.html is Blog Grid Page
  • blog-left-sidebar.html is Blog Left Sidebar Page
  • blog-right-sidebar.html is Blog Right Sidebar Page
  • blog-post.html is Blog Post Page
  • contact.html is Contact Us Page
  • event-details.html is Event Details Page
  • faq.html is FAQs Page
  • login.html is Login Page
  • photo-gallery.html is Photo Gallery Page
  • video-gallery.html is Video Gallery Page
  • search-results-grid-view.html is Search Results Grid View Page
  • search-results-list-view.html is Search Results List View Page
  • signup.html is Sign Up Page
  • team.html is Team Page
  • testimonial.html is Testimonial Page
